#+title: claude-code-ide-companion.el
#+author: Piotr Kwiecinski

Extra features for [[https://github.com/manzaltu/claude-code-ide.el][claude-code-ide.el]].

** Features

- *Project switch follow* — automatically shows the Claude Code session for the current project when switching projects via =project-switch-project=. If no session exists for the target project, any visible Claude Code windows are closed.

** Installation

This package is not yet available on MELPA. You can install it using one of the methods below.

*** Doom Emacs

Add to =packages.el=:

#+begin_src emacs-lisp
(package! claude-code-ide-companion
  :recipe (:host github :repo "piotrkwiecinski/claude-code-ide-companion.el"))
#+end_src

Add to =config.el=:

#+begin_src emacs-lisp
(use-package! claude-code-ide-companion
  :after claude-code-ide
  :config
  (claude-code-ide-companion-project-switch-mode 1))
#+end_src

*** use-package (with vc)

Emacs 29+ has built-in support for installing packages from VCS:

#+begin_src emacs-lisp
(use-package claude-code-ide-companion
  :vc (:url "https://github.com/piotrkwiecinski/claude-code-ide-companion.el"
        :rev :newest)
  :after claude-code-ide
  :config
  (claude-code-ide-companion-project-switch-mode 1))
#+end_src

*** use-package (with straight.el)

#+begin_src emacs-lisp
(use-package claude-code-ide-companion
  :straight (:host github :repo "piotrkwiecinski/claude-code-ide-companion.el")
  :after claude-code-ide
  :config
  (claude-code-ide-companion-project-switch-mode 1))
#+end_src

*** Manual

Clone the repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/claude-code-ide-companion.el")
(require 'claude-code-ide-companion)
(claude-code-ide-companion-project-switch-mode 1)
#+end_src

** Requirements

- Emacs 28.1+
- [[https://github.com/manzaltu/claude-code-ide.el][claude-code-ide]] 0.2.6+
